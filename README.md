# Pós-Graduação em Inteligência Artificial

## Repositório de Atividades, Projetos e Estudos

Este repositório reúne as atividades, exercícios, projetos, anotações e experimentos desenvolvidos durante a Pós-Graduação em Inteligência Artificial.

O objetivo é consolidar o conhecimento adquirido ao longo do curso, documentar a evolução do aprendizado e servir como base para futuros projetos profissionais e acadêmicos.
